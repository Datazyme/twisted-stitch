# Twisted Stitch

## Overview
A modern, responsive e-commerce knitware storefront designed for online shopping with a focus on payment integration and product presentation. Built using Next.js and Stripe and deployed on Vercel.

## Languages & Frameworks:
- Next.js (React)
- Tailwind CSS
- Stripe (payments)
- Vercel Deployment
- JavaScript

## Front-End Development:
- Interactive UI with Next.js, a React-based framework optimized for performance and SEO.
- Tailwind CSS for utility-first styling, enabling a responsive and consistent design system.

## Server-Side Rendering & Routing
- Next.js App Router and SSR/SSG features for optimized page delivery and SEO benefits.
- Integrated Stripe for secure checkout and payment processing.
